Instalador WPF

Visão Geral

Aplicação WPF (.NET) que configura o Edge PC Windows para rodar o sistema de contagem. Wizard de 4 etapas que instala dependências, configura variáveis de ambiente e registra o serviço Node.js.

Fluxo do Wizard

1

Verificação de Pré-requisitos

Checa Node.js 20+, conectividade de rede, permissões de administrador. Bloqueia avanço se faltar algo.

2

Configuração de Ambiente

Formulário para preencher variáveis: Supabase URL, Service Role Key, IPs das câmeras, porta do servidor. Validação em tempo real.

3

Instalação

Grava variáveis de ambiente (Machine-level), executa npm install, registra serviço Windows via NSSM.

4

Conclusão

Resumo do que foi configurado, status do serviço, botão para abrir o Monitor 1 no navegador.

Segurança

Variáveis sensíveis são gravadas como Machine-level environment variables (requer elevação de administrador). Nunca ficam em arquivo .env em produção.

Como Gerar o .exe

Pré-requisitos

  • • .NET SDK 8.0+ instalado
  • • Visual Studio 2022 ou dotnet CLI

Comando

dotnet publish -c Release -r win-x64 --self-contained true /p:PublishSingleFile=true

O executável gerado fica em bin/Release/net8.0-windows/win-x64/publish/

Como Instalar

1

Copiar o .exe para o Edge PC de destino

2

Executar como Administrador (botão direito → Executar como administrador)

3

Seguir o wizard: preencher variáveis, aguardar instalação

4

Verificar que o serviço rei-contagem está rodando no Gerenciador de Serviços

Quando Regerar

Precisa regerar o .exe

  • • Nova variável de ambiente adicionada
  • • Mudança no fluxo do wizard
  • • Atualização de dependência .NET
  • • Correção de bug no instalador

Não precisa regerar

  • • Mudanças no código Node.js
  • • Novas migrations SQLite
  • • Alterações no frontend
  • • Atualizações de dependências npm
