API REST

Sessões

GET
/sessoes

Lista sessões (filtro por embarque, status)

POST
/sessoes

Abre nova sessão. Body: numero_embarque, codigo_op, codigo_operador, camera_id

POST
/sessoes/:id/confirmar

Confirma programa da câmera para a sessão

POST
/sessoes/:id/encerrar

Encerra sessão e emite etiqueta automaticamente

Etiquetas

POST
/etiquetas/caixas

Reimpressão manual. Body: numero_embarque, numero_caixa, codigo_operador

GET
/etiquetas/caixas?embarque=X&caixa=Y

Histórico de emissões da caixa

POST
/etiquetas/:id/retry

Recoloca partes com erro na fila de impressão

Faturamento

POST
/faturamento/sessoes/:id/aprovar

Aprova sessão. Body: codigo_aprovador

POST
/faturamento/sessoes/:id/reprovar

Reprova sessão. Body: codigo_aprovador

POST
/faturamento/sessoes/:id/realocar

Realoca sessão para outro embarque. Body: embarque_destino

Controle de Acesso

GET
/acesso/catalogo

Catálogo de atividades agrupado por página

GET
/acesso/grupos

Lista grupos com atividades atribuídas

POST
/acesso/grupos

Cria grupo. Body: nome, descricao?

PATCH
/acesso/grupos/:id

Renomeia grupo. Body: nome?, descricao?

DELETE
/acesso/grupos/:id

Exclui grupo (cascade em atividades e vínculos)

PUT
/acesso/grupos/:id/atividades

Define atividades do grupo. Body: { atividades: [...] }

GET
/acesso/usuarios

Lista usuários (sincroniza do Supabase auth se online)

PUT
/acesso/usuarios/:id/grupos

Atribui grupos ao usuário. Body: { grupos: [...] }

PUT
/acesso/usuarios/:id/overrides

Define overrides. Body: { overrides: [{ atividade_id, efeito }] }

GET
/acesso/usuarios/:id/acesso

Retorna grupos, overrides e acesso efetivo calculado

Embarques, OPs e Operadores

GET
/embarques

Lista embarques abertos

GET
/ops

Lista ordens de produção

GET
/operadores

Lista operadores ativos

GET
/health

Estado do runtime: sync, outbox, câmeras
