Fluxo de Emissão

  1. Operador encerra sessão e escolhe caixa
  2. sessaoService.encerrar() grava sessão
  3. caixaLabelService.emitirPorEncerramento() monta documento lógico
  4. Renderer ZPL gera payload(s) com paginação
  5. Tabelas etiquetas_caixa + etiquetas_caixa_partes recebem dados
  6. Print queue envia via TCP (porta 9100) ou spooler
  7. Falha de impressão NÃO reverte a sessão

  • src/labels/caixa-label-service.js — monta documento, persiste, enfileira
  • src/printer/print-queue.js — fila de impressão com retry
  • src/printer/transports/tcp.js — envio TCP raw
